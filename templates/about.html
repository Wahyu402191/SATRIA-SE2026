SATRIA SE2026

Scraping and Sentiment Analysis Tracker for Economic Census 2026 — Sistem analisis sentimen komentar media sosial YouTube untuk mendukung program Sensus Ekonomi BPS Kabupaten Bangkalan.

Versi 1.0  |  BPS Kabupaten Bangkalan  |  2026
Metode Analisis Sentimen
1

Naive Bayes

Probabilistic classifier berbasis TF-IDF dengan trigram features. Training menggunakan 1.200 sampel dari lexicon Indonesia dengan alpha 0.5 (Laplace smoothing).

2

Support Vector Machine

Linear SVM dengan parameter C=1.5 dan class weight balanced. Efektif untuk klasifikasi teks dengan dimensi fitur tinggi.

3

LSTM

Long Short-Term Memory dengan context window dan weighted sequence scoring. Mempertimbangkan urutan kata dan konteks kalimat.

4

IndoBERT

Simulasi bidirectional transformer untuk Bahasa Indonesia. Menggunakan attention mechanism dan negation handling kontekstual.
